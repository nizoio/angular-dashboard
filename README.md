## Local Development
1. Download this project
2. Install NodeJs from [NodeJs Official Page](https://nodejs.org/en).
3. Open Terminal
4. Go (cd) to the project folder
5. Run in terminal: ```npm install -g @angular/cli```
6. Then: ```npm install```
7. Finally run: ```ng serve -o```
